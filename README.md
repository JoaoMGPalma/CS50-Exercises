CS 50 - Problem sets (Lectures 1-3)
 
 This repository contains my personal implementations of several CS50 problem sets, rewritten in
portable C and adapted for environments outside the CS50 IDE. 
 I've changed some implementation so that the CS50 library is not necessary for 
testing, all while avoiding changes in the code style i had at the time so some evolution can be 
percieved when browsing other code in my Reporitory.

📂 What’s inside
- Credit Card Validator — Luhn’s algorithm + network detection
- Mario Pyramids — Loop logic and formatted output
- Substitution Cipher — Command‑line key validation and text encryption
- Tideman (Ranked Pairs) — Graphs, cycle detection, and sorting
